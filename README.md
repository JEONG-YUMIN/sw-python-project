# 시험 D-day와 공부 시간 관리 프로그램

## 무엇을 하는 프로그램인가요?

이 프로그램은 시험 기간에 여러 과목의 시험 날짜와 공부 시간을 관리하기 위한 프로그램입니다. 사용자는 과목명, 시험 날짜, 목표 공부 시간, 현재까지 공부한 시간을 입력할 수 있습니다. 프로그램은 시험까지 남은 D-day, 목표 대비 공부 진행률, 부족한 공부 시간, 하루 평균 필요한 공부 시간을 계산해줍니다. 또한 부족한 공부 시간과 시험까지 남은 날짜를 기준으로 현재 우선적으로 공부해야 할 과목을 추천합니다.

## 실행 방법

1. Python 3.10 이상을 설치합니다.
2. vs code 프로젝트 폴더에서 터미널을 실행합니다.
3. 실행 버튼을 눌러 프로젝트를 실행합니다.
4. 프로그램이 실행되면 메뉴 번호를 입력하여 기능을 사용할 수 있습니다.

## 주요 기능

* 과목 등록
* 오늘 공부 시간 추가
* 공부 현황 보기
* 등록된 과목 목록 보기
* 시험까지 남은 D-day 계산
* 공부 진행률 출력
* 우선 공부 과목 추천
* JSON 파일을 이용한 데이터 저장 및 불러오기

## 사용된 Python 개념

* 변수
* 조건문
* 반복문
* 함수
* 리스트

## AI 도구 사용 내역

ChatGPT를 사용하여 코드 구조 작성, 오류 확인, 발표 준비에 도움을 받았습니다. 

특히 과목 정보를 저장하는 구조, JSON 파일 저장 방식, D-day 계산 방식, 추천 과목 기준을 정리하는 과정에서 AI의 도움을 받았습니다. 이후 프로그램을 직접 실행하면서 과목 등록, 공부 시간 추가, 공부 현황 출력, 데이터 저장 기능이 정상적으로 동작하는지 확인했습니다.

## 한계와 다음 단계

현재 프로그램은 터미널에서 숫자를 입력하는 CLI 방식으로 실행됩니다. 따라서 버튼이나 입력창이 있는 화면형 프로그램보다 사용성이 제한적입니다. 또한 등록한 과목을 수정하거나 삭제하는 기능은 아직 구현하지 못했습니다.

다음 단계에서는 과목 수정 및 삭제 기능을 추가하고 싶습니다. 또한 tkinter를 사용한 화면형 프로그램으로 개선하여 버튼과 입력창을 통해 더 직관적으로 사용할 수 있도록 만들고 싶습니다. 추가로 날짜별 공부 기록을 저장하여 공부 패턴을 확인할 수 있는 기능도 구현해보고 싶습니다.
