# 프로젝트제안서
## 1. 프로젝트 이름
시험 D-day와 과목별 공부 시간 관리 프로그램
## 2. 한문장 설명
시험일까지 남은 날짜를 계산하고 과목별 목표 공부 시간과 실제 공부 시간을 비교해서 부족한 과목을 알려주는 프로그램이다.
## 3. 풀려는 문제 또는 만드려는 이유
시험 기간에는 여러 과목을 동시에 공부해야 해서 계획을 세우기가 어렵다.
또한 시험 공부만으로 벅찬 대학생들에게 시험 일정 관리는 또 다른 스트레스로 다가온다. 그래서 시험 날짜와 과목별 공부 시간을 입력하면 남은 날짜와 부족한 과목을 한눈에 확인할 수 있는 프로그램을 만들고자 한다.
## 4. 핵심 기능 3가지
1. 시험 날짜를 입력하면 오늘 날짜 기준으로 시험까지 남은 날짜를 계산한다.
2. 과목별 목표 공부 시간과 실제 공부 시간을 입력받는다.
3. 목표보다 공부 시간이 부족한 과목을 찾아서 우선적으로 공부할 과목울 추천한다.
## 5. 예상 사용자
시험 기간에 여러 과목을 동시에 공부해야 하는 대학생
## 6. 결과물 형태
CLI 프로그램
## 7. 사용할 python 개념
변수, 조건문, 반복문, 함수, 리스트, 딕셔너리, 문자열 처리, 날짜 계산, 기본 연산
